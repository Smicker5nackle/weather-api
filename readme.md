# ☁️ WeatherNow — Live Weather Updates

A beautiful, modern weather app that provides real-time weather information for any location worldwide. Built with a stunning glassmorphism UI and smooth animations.

## ✨ Features

- 🔍 **City Search** — Search by city name, zip code, or coordinates
- 📍 **GPS Location** — One-click weather for your current location
- 🌡️ **Detailed Weather Data** — Temperature, humidity, wind speed, pressure, visibility, UV index & cloud cover
- 🌿 **Air Quality Index** — Visual AQI bar with EPA rating
- 🎨 **Aurora UI** — Glassmorphism design with animated aurora blobs
- 📱 **Responsive** — Works seamlessly on desktop and mobile

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, CSS animations
- **JavaScript (Vanilla)** — Async/Await, Fetch API
- **WeatherAPI** — Real-time weather data provider

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/Smicker5nackle/weather-api.git
   ```
2. Open `index.html` in your browser — that's it! No build tools needed.

## 📸 How It Works

1. Enter a city name in the search bar (or click the GPS button)
2. Hit **"Get Weather"**
3. View real-time temperature, conditions, and weather details
4. Check the Air Quality Index at the bottom

This app uses [WeatherAPI](https://www.weatherapi.com/) to fetch real-time weather data including current conditions and air quality information.
